How to Use Your Camera Scanner System

📸 Camera Usage:

  1. Navigate to /camera or click "Camera Scanner" from dashboard
  2. Click "Activate Camera" to start live video stream
  3. Position inventory items within camera view
  4. Use "Stop Camera" to end session
  5. Alternative: Use "Upload Image" for existing photos

📱 Scanner Usage:

  1. Click "Activate Scanner" to start barcode detection
  2. System simulates scanning process (2-second delay)
  3. View scanned results in "Recent Scans" section
  4. Click "Add to Inventory" to process scanned items
  5. Use "Batch Scan Mode" for multiple items

🔄 Workflow Integration:

  1. Start from Dashboard → Camera Scanner
  2. Scan or photograph inventory items
  3. Process scanned data
  4. Return to Dashboard with new inventory data
  5. Voice alerts work with new items automatically
